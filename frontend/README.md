# Shortly — URL Shortener Frontend

Premium dark-mode URL shortener frontend built with React + TypeScript + Vite + Tailwind.

## Setup

```bash
npm install
npm run dev
```

Backend is expected on `http://localhost:5000`.

## Features
- Landing, Register, Login
- Protected Dashboard with stats + charts
- My Links with search/filter/sort/pagination
- Create / Edit / Soft-delete links
- Glassmorphism dark UI, indigo→purple gradient accents
- JWT stored in localStorage under `token`
